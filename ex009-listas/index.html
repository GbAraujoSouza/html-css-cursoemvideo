Listas em HTML

Listas ordenadas

Podemos criar uma lista ordenada usando a tag <ol> (ordered list)

  1. Top 1
  2. Top 2
  3. Top 3
  4. Top 4

Listas não Ordenadas

Podemos criar listas não ordenadas usando a tag <ul> (unordered list)

  • item 1
  • item 2
  • item 3
  • item 4

Listas Mistas

Para criar listas mistas basta adicionar uma tag de lista dentro da outra

  1. item a
  2. item b
    1. item b.1
    2. item b.2
  3. item c
